# Integrations

Tunnet connects to the tools you already run. Integrations install alongside your stack and join those environments to your mesh - without replacing how you deploy apps today.

## Available integrations

| Integration | What it does |
| --- | --- |
| [Kubernetes](/integrations/kubernetes/) | Join a cluster to your mesh, advertise pod/Service CIDRs, and expose cluster services with Tunnet Serve or Tunnel |
| [Terraform / OpenTofu](/guide/policy-as-code#terraform-and-opentofu) | Manage policy with granular resources or a monolithic document; export live state into Terraform |
| [GitOps CI](/guide/policy-as-code#gitops) | Validate, simulate, and apply policy from GitHub Actions, GitLab CI, or Bitbucket Pipelines |

More integrations will land here as they ship.
