# tunnet

Unscoped alias for [`@tunnet/sdk`](https://www.npmjs.com/package/@tunnet/sdk).

```bash
npm install tunnet
# same as
npm install @tunnet/sdk
```

```ts
import { enroll, TunnetNode } from "tunnet";
```

Prefer `@tunnet/sdk` in new projects. This package exists so `tunnet` on npm resolves to the same SDK.
